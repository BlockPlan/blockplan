<task type="auto">
  <name>Task 1: Create upload URL and extraction API routes</name>
  <files>
    app/api/syllabi/upload-url/route.ts
    app/api/syllabi/extract/route.ts
  </files>
  <action>
    Create `app/api/syllabi/upload-url/route.ts`:
    - POST handler accepting JSON `{ courseId, filename }`
    - Validate with `extractRequestSchema` pattern (courseId is UUID, filename is non-empty)
    - Authenticate with `supabase.auth.getUser()` — return 401 if no user
    - Verify courseId belongs to user: query `courses` table with `.eq('id', courseId).eq('user_id', user.id).single()`
    - Return 403 if course not found
    - Validate filename: must end with `.pdf`, max 255 chars
    - Construct storage path: `${user.id}/${courseId}/${Date.now()}-${filename}`
    - Call `supabase.storage.from('syllabi').createSignedUploadUrl(storagePath)`
    - Return JSON `{ signedUrl: data.signedUrl, path: storagePath, token: data.token }`

    Create `app/api/syllabi/extract/route.ts`:
    - POST handler accepting JSON `{ storagePath, courseId }`
    - Authenticate with `supabase.auth.getUser()` — return 401 if no user
    - SECURITY: validate storagePath starts with `${user.id}/` — return 403 otherwise (prevents accessing other users' PDFs)
    - Verify courseId belongs to user (same query as upload-url)
    - Call `extractSyllabusText(storagePath)` from `@/lib/syllabus/extract`
    - If `result.isEmpty`: return JSON `{ error: 'no-text', message: 'This PDF appears to be a scanned image. It cannot be parsed automatically. Please enter your assignments manually.' }` with status 422
    - Get term for the course: query `courses` join `terms` to get `terms.start_date`
    - Call `parseWithRules(result.text, new Date(term.start_date))` from `@/lib/syllabus/parser-rule-based`
    - Call `parseWithLLM(result.text, \`Term: ${term.name}, starts ${term.start_date}\`)` from `@/lib/syllabus/parser-llm`
    - Call `mergeParserResults(ruleItems, llmItems)` from `@/lib/syllabus/parser-merge`
    - Return JSON `{ items: mergedItems, totalPages: result.totalPages, llmUsed: llmItems.length > 0 }`
    - Wrap in try/catch — on error return 500 with `{ error: 'extraction-failed', message: 'Failed to process syllabus. Please try again or enter tasks manually.' }`
  </action>
