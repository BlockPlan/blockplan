**Install sonner:** `npm install sonner`

**Update app/layout.tsx:**
- Import `Toaster` from 'sonner'
- Add `<Toaster position="bottom-right" />` inside the body, after `{children}`

**Create app/plan/actions.ts** ('use server'):

1. **generatePlan()** Server Action:
   - Auth check via supabase.auth.getUser()
   - Load incomplete tasks: supabase.from('tasks').select('id, title, course_id, due_date, estimated_minutes, status').eq('user_id', user.id).neq('status', 'done')
   - Load availability rules: supabase.from('availability_rules').select('day_of_week, start_time, end_time, rule_type').eq('user_id', user.id)
   - Load user profile (planner_settings, timezone): supabase.from('user_profiles').select('planner_settings, timezone').eq('id', user.id).single()
   - If no profile exists, use DEFAULT_PLANNER_SETTINGS from lib/validations/planner.ts and 'America/Boise' as fallback timezone
   - Auto-detect timezone: If no profile exists and browser timezone was passed in formData, create profile with that timezone
   - Call generateSchedule() from lib/services/scheduler.ts
   - Delete existing scheduled (not done/missed) plan_blocks: supabase.from('plan_blocks').delete().eq('user_id', user.id).eq('status', 'scheduled')
   - Insert new blocks: supabase.from('plan_blocks').insert(result.blocks.map(...)) with user_id, task_id, start_time (ISO), end_time (ISO), status='scheduled'
   - revalidatePath('/plan')
   - Return { success: true, blocksScheduled: result.blocks.length, unscheduledTaskIds: result.unscheduled.map(t => t.id), riskTasks: result.risk_tasks.map(rt => ({ taskId: rt.task.id, taskTitle: rt.task.title, level: rt.level })) }

2. **markBlockDone(blockId: string)** Server Action:
   - Auth check
   - Update plan_blocks set status='done' where id=blockId and user_id=user.id
   - revalidatePath('/plan')
   - Return { success: true }

3. **markBlockMissed(blockId: string)** Server Action:
   - Auth check
   - Update plan_blocks set status='missed' where id=blockId and user_id=user.id
   - Then trigger auto-reschedule: run the same logic as generatePlan() but with planStart=new Date() (now) and only delete future 'scheduled' blocks
   - revalidatePath('/plan')
   - Return { success: true, rescheduledCount: newResult.blocks.length }

Extract shared scheduling logic into a private helper function `_runScheduler(supabase, userId)` to avoid duplication between generatePlan and markBlockMissed.
