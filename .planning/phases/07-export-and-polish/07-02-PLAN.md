Create lib/validations/subtask.ts:
- Zod schema for subtask creation:
  ```typescript
  import { z } from "zod";
  export const subtaskSchema = z.object({
    task_id: z.string().uuid(),
    title: z.string().min(1).max(200),
    due_date: z.string().optional(),
    estimated_minutes: z.number().int().min(1).max(1440),
    sort_order: z.number().int().min(0),
  });
  export type SubtaskInput = z.infer<typeof subtaskSchema>;
  ```

Create lib/services/subtask-suggestions.ts:
- Import subDays, isAfter, startOfDay from date-fns
- Export interface SubtaskSuggestion: { title: string; due_date: Date; estimated_minutes: number; sort_order: number }
- Export function generateSubtaskSuggestions(parentDueDate: Date, parentEstimatedMinutes: number): SubtaskSuggestion[]
- Returns 4 subtasks:
  - "Outline": due 10 days before parent, 15% of estimated time, sort_order 0
  - "First Draft": due 7 days before parent, 40% of estimated time, sort_order 1
  - "Revise": due 3 days before parent, 30% of estimated time, sort_order 2
  - "Final Submit": due on parent due date, 15% of estimated time, sort_order 3
- Clamp: if any subtask due_date would be before today (startOfDay(new Date())), clamp to today
- Round estimated_minutes to nearest integer using Math.round
- Export function shouldSuggestSubtasks(type: string, estimatedMinutes: number): boolean
  - Returns true only when type === "assignment" AND estimatedMinutes >= 240

- Props: { taskId: string; taskDueDate: string; taskEstimatedMinutes: number; onClose: () => void }
